# MarqGo: ระบบบริหารจัดการข้อมูลร้านค้าตลาดนัด

## รายละเอียด

Mobile Solution ที่ช่วยให้ผู้ซื้อค้นหาพิกัดร้านค้าและเมนูอาหารในตลาดนัด พร้อมระบบช่วยตัดสินใจ "วันนี้กินอะไรดี"

---

## แรงบันดาลใจ (Inspiration)

การเดินตลาดนัดมักพบปัญหา ความไม่แน่นอน ของตำแหน่งร้านค้าและการเปิด-ปิดในแต่ละวัน รวมถึงปัญหาคลาสสิกของผู้บริโภคคือ "ไม่รู้จะกินอะไรดี" 

ผมจึงสนใจพัฒนา MarqGo เพื่อเป็นแพลตฟอร์มที่เชื่อมโยงพ่อค้าแม่ค้ากับผู้ซื้อเข้าด้วยกัน โดยมีจุดเด่นดังนี้:

* Digital Profile: ร้านค้ามีตัวตนบนโลกออนไลน์
* Live Location: ระบุตำแหน่งล็อกขายที่แน่นอนในแต่ละวัน
* Real-time Update: อัปเดตสถานะการเปิดร้านและเมนูประจำวัน

---

## แนวคิดและการออกแบบ (Concept & Design)

ระบบถูกออกแบบโดยแบ่งออกเป็นสองส่วนหลัก (Buyer & Seller Apps) โดยเน้นไปที่เวอร์ชัน MVP สำหรับฝั่งผู้ซื้อเป็นอันดับแรก:

### ฟีเจอร์เด่น (Key Features)

1.  Interactive Market Map: แสดงพิกัดร้านค้าที่เปิดให้บริการจริงผ่านแผนที่จำลอง

2.  Random Menu Game: นำเมนูจากร้านค้าต่างๆ มาสุ่มเพื่อช่วยแก้ปัญหาการตัดสินใจเลือกซื้ออาหาร

3.  Smart Navigation: เมื่อได้เมนูที่ต้องการ ระบบจะนำทางไปยังพิกัดร้านนั้นๆ ทันที
